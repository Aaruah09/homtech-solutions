## How to run

1. Clone the repo:
   ```bash
   git clone https://github.com/YOUR_USERNAME/homtechsolutions.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the dev server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
